# Python for Data Analysis — 30 Day Challenge

A public 30-day challenge to build real data analysis skills with Python. Two tracks so anyone can join, regardless of experience level.

---

## Quick Start

1. **Pick your track** in [30-DAY-ROADMAP.md](30-DAY-ROADMAP.md):
   - **Track A** — you already know Python (intermediate)
   - **Track B** — you know programming but are new to Python

2. **Set up your environment:**
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   pip install numpy pandas matplotlib seaborn jupyterlab streamlit
   ```

3. **Start Day 1** and work through at your own pace.
